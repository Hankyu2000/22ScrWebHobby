使用するプログラミング言語(+α)

言語の使用目的を紹介

言語 使用目的
(HTML) 実はプログラミング言語じゃない。Webページの構造化、HTML内CSS、ダメ、ゼッタイ。（セマンティックHTML的に）
Javascript 今やWebの領域を飛び出して（主にNodeJSのせい）かなり広範囲で活躍してる感じの言語。
CSS Webページ作るのにHTMLの次に重要。これ無しではWeb開発はほぼほぼ何もできない。
Python ZKDevの標準開発言語。機械学習から数理計算まで大きな数が絡むところで活躍。何よりコードが美しい。
PHP かつて標準開発言語として使われていた。Webだけでしか使えないことから動的Web言語としてはBlazor系に道を譲った。
C 今やC++やC#など多くの派生系がいるものの、ZKDevではC言語自体は余り使われない。
C# ZKDevのもう一つの標準開発言語。Unityとかの3Dゲーム開発からBlazor系でのWeb開発まで幅広くこなしてくれる。
